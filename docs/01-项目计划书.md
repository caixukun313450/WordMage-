# WordMage 单词魔法师 — 项目计划书

## 1. 项目背景

面向中国小学 3-6 年级学生的英语词汇学习存在"记词枯燥、缺乏情境"的痛点。本项目旨在利用大语言模型生成适龄英文小故事，结合游戏化激励与本地学习记录，构建可运行的 Web 学习原型。

## 2. 项目目标

1. 构建可运行的 Web 原型，核心功能：输入 1-5 个单词，大模型返回英文小故事，并配中文翻译及 TTS 语音。
2. 实现极简游戏化元素：每日生成故事可收集星星，累积 10 颗星解锁"单词宝藏"彩蛋。
3. 搭建简易学习分析：记录最近 7 天生成的单词，用本地存储展示复习列表。
4. 输出全套软件工程文档，满足实训要求。

## 3. 项目范围（MVP）

| 模块 | 内容 |
|------|------|
| 情境生成器 | 集成智谱 AI / 通义千问，生成故事并高亮关键词 |
| 单词输入 | 200 词基础词库 + 每日 3 个推荐词 |
| 语音朗读 | 浏览器 SpeechSynthesis |
| 复习闯关 | 每 3 个故事触发单词配对游戏 |
| 学习记录 | localStorage 保存历史与词频 |

## 4. 技术选型

- **后端**：Spring Boot 4 + Java 21，提供 REST API 代理大模型调用
- **前端**：原生 HTML/CSS/JS，无构建依赖，便于部署
- **存储**：localStorage（客户端），无需数据库
- **大模型**：智谱 GLM-4-Flash（默认），可切换通义千问

## 6. 风险与对策

| 风险 | 对策 |
|------|------|
| API 额度不足 | 提供演示模式；支持多厂商切换 |
| 浏览器 TTS 兼容性 | 降级提示；推荐使用 Chrome/Edge |
| 故事内容不适龄 | Prompt 约束年级与积极主题 |
