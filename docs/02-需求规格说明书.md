# WordMage 需求规格说明书（SRS）

**版本**：1.0  
**日期**：2026-06-10

## 1. 引言

### 1.1 目的

本文档描述 WordMage 单词魔法师 Web 原型的功能与非功能需求，作为开发、测试与验收的依据。

### 1.2 范围

面向小学生的英语单词情境学习工具，MVP 为浏览器 Web 应用。

## 2. 功能需求

### FR-01 单词输入

- 用户可手动输入 1-5 个英文单词
- 提供 200 词小学高频词库，点击添加
- 每日固定推荐 3 个"魔法单词"（基于日期种子）

### FR-02 故事生成

- 调用大模型 API 生成 80-120 词英文小故事
- 同时返回中文翻译
- 故事中输入单词需高亮显示
- 未配置 API Key 时使用演示模板

### FR-03 语音朗读

- 支持朗读英文故事正文
- 支持朗读中文翻译
- 使用浏览器原生 SpeechSynthesis

### FR-04 游戏化 — 星星

- 每日首次成功生成故事获得 1 颗星
- 界面显示当前星星数 / 10

### FR-05 游戏化 — 单词宝藏

- 累积 10 颗星解锁彩蛋弹窗
- 展示学习成就与隐藏词汇

### FR-06 复习闯关

- 每累计生成 3 个故事，自动弹出单词配对小游戏
- 从已学单词中抽取英文与中文进行配对

### FR-07 学习记录

- localStorage 保存故事生成历史
- 复习列表：展示最近 7 天出现的单词及出现次数
- 学习报告：故事总数、单词总数、星星数、词频 Top 10

## 3. 非功能需求

| 编号 | 需求 | 指标 |
|------|------|------|
| NFR-01 | 响应时间 | 故事生成 ≤ 15s（含网络） |
| NFR-02 | 兼容性 | Chrome、Edge 最新版 |
| NFR-03 | 隐私 | 学习数据仅存本地，不上传服务器 |
| NFR-04 | 可用性 | 界面简洁，适合 8-12 岁儿童操作 |

## 4. 接口需求

### POST /api/story

**请求**：
```json
{ "words": ["apple", "happy"] }
```

**响应**：
```json
{
  "title": "A Happy Apple Day",
  "storyEn": "...",
  "storyZh": "...",
  "words": ["apple", "happy"],
  "demoMode": false
}
```

## 5. 约束

- 单词数量：1-5 个
- 目标用户：小学 3-6 年级
- 先期使用智谱 AI 或通义千问免费额度
