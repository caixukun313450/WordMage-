# 免绑卡公网访问指南

Render 需要绑信用卡，以下方案**完全免费、无需绑卡**。

---

## 方案 A：Cloudflare 隧道（推荐，5 分钟）

把本机运行的 WordMage 映射到公网，适合答辩演示。

### 第 1 步：安装 cloudflared

PowerShell 执行：

```powershell
winget install Cloudflare.cloudflared
```

安装后**关闭并重新打开** PowerShell。

若 winget 不可用，手动下载：
https://github.com/cloudflare/cloudflared/releases/latest  
下载 `cloudflared-windows-amd64.exe`，重命名为 `cloudflared.exe`，放到 `deploy` 目录。

### 第 2 步：启动 WordMage

在 IntelliJ 里正常运行项目（端口 8080）。

### 第 3 步：开启公网隧道

```powershell
cd D:\WordMage\WordMage
.\deploy\start-public.ps1
```

或直接：

```powershell
cloudflared tunnel --url http://localhost:8080
```

### 第 4 步：复制公网地址

终端里会出现类似：

```
https://random-words.trycloudflare.com
```

把这个链接发给同学即可访问。

### 注意

| 项目 | 说明 |
|------|------|
| 电脑需保持开机 | 关机会无法访问 |
| 地址每次变化 | 重启隧道后链接会变 |
| 免费无绑卡 | 无需注册 Cloudflare 账号 |

---

## 方案 B：cpolar 内网穿透（国内稳定）

适合需要固定域名、国内访问更稳的场景。

1. 注册 https://www.cpolar.com/ （微信/手机号）
2. 下载 Windows 客户端并安装
3. 登录后添加隧道：
   - 本地地址：`http://localhost:8080`
   - 类型：http
4. 启动隧道，获得 `https://xxxx.cpolar.cn` 公网地址

免费版有流量和隧道数限制，演示够用。

---

## 方案 C：同一 WiFi 局域网（零配置）

你和同学在**同一 WiFi** 下：

```
http://10.12.146.234:8080
```

无需任何工具，但仅限同一网络。

## 方案对比

| 方案 | 绑卡 | 外网访问 | 难度 |
|------|------|----------|------|
| Cloudflare 隧道 | 否 | 是 | ⭐ |
| cpolar | 否 | 是 | ⭐⭐ |
| 局域网 | 否 | 仅同 WiFi | ⭐ |
| Render | 需要 | 是 | ⭐⭐ |

---

## 答辩推荐流程

1. 提前在 IntelliJ 启动 WordMage
2. 运行 `.\deploy\start-public.ps1`
3. 把生成的 `https://xxxx.trycloudflare.com` 链接投屏或发群里
4. 答辩结束后按 Ctrl+C 关闭隧道
