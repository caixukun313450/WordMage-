## 3. 接口测试

```bash
curl -X POST http://localhost:8080/api/story \
  -H "Content-Type: application/json" \
  -d "{\"words\":[\"cat\",\"dog\"]}"
```

响应包含 title、storyEn、storyZh、words 字段。

## 5. 已知问题

1. 演示模式下故事模板固定，多样性有限
2. Safari 对 SpeechSynthesis 支持因系统而异
3. 词库存在少量重复词条（不影响功能）

## 6. 结论

MVP 核心功能全部通过测试，满足需求规格说明书中的功能需求，可交付演示与实训验收。
