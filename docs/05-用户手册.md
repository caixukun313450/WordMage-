# WordMage 用户手册

**版本**：1.0  
**适用对象**：小学生及家长、教师

## 1. 开始使用

1. 确保 WordMage 服务已启动（见 README）
2. 打开浏览器，访问 http://localhost:8080
3. 推荐使用 Chrome 或 Edge 以获得最佳语音朗读效果

## 2. 创作魔法故事

### 2.1 选择单词

- **每日魔法单词**：页面顶部每天会推荐 3 个单词，点击即可添加
- **手动输入**：在输入框中输入英文单词，按回车或点击"添加"
- **词库选择**：点击"打开词库"，从 200 个常用词中点击添加
- 最多同时选择 **5 个**单词

### 2.2 生成故事

1. 选好单词后，点击 **"生成魔法故事"**
2. 等待几秒，英文故事和中文翻译会出现在下方
3. 你输入的单词会在故事中 **黄色高亮** 显示

### 2.3 听故事

- 点击 **"朗读英文"** 听英文故事
- 点击 **"朗读中文"** 听中文翻译

## 3. 收集星星

- 每天 **第一次** 成功生成故事，可获得 **1 颗星星** ⭐
- 右上角显示当前星星数量
- 收集满 **10 颗星**，会解锁 **"单词宝藏"** 彩蛋 🎁

## 4. 单词配对闯关

- 每生成 **3 个故事**，会自动弹出配对小游戏
- 点击英文单词，再点击对应的中文释义完成配对
- 全部配对成功即可过关

## 5. 复习与学习报告

### 复习列表

- 切换到 **"复习列表"** 标签
- 查看最近 7 天学过的单词
- 点击单词可朗读发音

### 学习报告

- 切换到 **"学习报告"** 标签
- 查看：生成故事数、学习单词数、收集星星数
- 查看词频 Top 10，了解哪些词练习最多

## 6. 常见问题

**Q：故事内容每次都一样？**  
A：若未配置大模型 API Key，系统处于演示模式。请按 README 配置智谱或通义千问 API Key。

**Q：听不到声音？**  
A：检查浏览器是否允许网页播放声音，并确认系统音量已打开。

**Q：数据会丢失吗？**  
A：学习记录保存在浏览器本地。清除浏览器数据会导致记录丢失。

**Q：可以在手机上用吗？**  
A：可以，页面已适配移动端，但 TTS 效果因手机浏览器而异。
